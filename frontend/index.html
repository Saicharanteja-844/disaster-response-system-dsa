About This Project

Overview

This project implements a real-time Disaster Response & Rescue Command Center applying Graph Data Structures & Algorithms. It calculates paths and identifies resources under simulated road blockages and hazard level conditions.

Key Algorithms

  • Dijkstra's Algorithm: Computes safety-balanced shortest paths by minimizing physical distance scaled with a danger index. Complexity: O(E log V).
  • A* Search Algorithm: Enhances Dijkstra's using a goal-oriented spatial heuristic calculated via the Haversine formula. Complexity: O(E log V).
  • BFS (Breadth-First Search): Computes route based on minimum hop count (fewest edges). Complexity: O(V + E).
  • DFS (Depth-First Search): Recursively explores search depth paths. Complexity: O(V + E).

Backend Implementation

Built using high-performance C++ for adj list graph storage, priority queues, and sorting operations. Program stdout is returned as JSON strings to an Express server bridge.
